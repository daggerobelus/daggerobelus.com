# Lorraine Witch Trials Database

Analysis of 368 witch trial records from the Lorraine region of France, spanning the 16th-17th centuries. This project applies social network analysis to one of Europe's richest witch trial archives.

## Data Overview

- **368** digitized trial records (PDFs)
- **157** trials extracted to structured JSON
- **600+** persons identified
- **2,000+** relationships mapped
